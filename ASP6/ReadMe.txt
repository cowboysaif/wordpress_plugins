REQUIREMENTS
-------------------
1. fopen enabled.
2. Curl enabled, with openSSL.
3. WordPress 2.3+ installed.
4. ionCube support, loaders are available at http://www.ioncube.com/loaders.php.


INSTALLATION
--------------------
* Upload the contents of the zip folder to your /wp-content/plugins/ folder.

* Set permission to 777 for the plugin's "reports" folders.

* ASP requires ioncube loaders to be installed on your server. Such loaders are required for Ioncube encoded files. 
Since ioncube is widely used, chances is that your server already has ionCube support already enabled. 
If this is not the case, you would need to install it yourself or ask your web host as they will surely be able to help.
For more information on ioncube, please visit: http://www.ioncube.com/loaders.php
For a tutorial on how to install ioncube loaders on a bluehost server, please see: http://helpdesk.bluehost.com/kb/index.php?x=&mod_id=2&id=171
For installation instructions specific to your server, don't hesitate to ask your web host.


UPGRADATION
-------------------
* Upload and overwrite the contents of the zip folder to your /wp-content/plugins/ folder.

* First deactivate and then re-activate the plugin.


ACTIVATE THE PLUGIN
--------------------
Activate your plugin by going to your PLUGINS tab and choosing to activate the plugin called Auto Social Poster.

Next, enter your license key and add your account information at OPTIONS --> AUTO SOCIAL POSTER
